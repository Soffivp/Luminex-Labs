<h1>Modelo de Datos</h1>

<h3>DIAGRAMA DE MODELO DE DATOS – SISTEMA DE BOLSA DE EMPLEO CAIL</h3>

<p>
  El modelo relacional es universalmente reconocido en la academia y la industria,
  permitiendo visualizar con claridad las entidades del negocio, sus atributos y
  las relaciones que las conectan. Sin embargo, la implementación técnica real
  utilizará <strong>Firebase Firestore</strong>, la cual es una base de datos
  <strong>NoSQL orientada a documentos</strong>, que ofrece ventajas significativas
  para el sistema como:
</p>

<ul>
  <li>Escalabilidad automática sin intervención manual</li>
  <li>Sincronización de datos en tiempo real para notificaciones instantáneas</li>
  <li>Modelo de datos flexible que reduce la complejidad de consultas</li>
  <li>Integración nativa con servicios de autenticación y almacenamiento en la nube</li>
  <li>Modelo de costos optimizado para aplicaciones en crecimiento</li>
</ul>
